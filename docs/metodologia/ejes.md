# Ejes de medición

| Eje | Qué mide | Escala |
|-----|----------|--------|
| Integridad | Solidez y consistencia probatoria | 0–10 |
| Sincronía | Correlación temporal con eventos políticos | 0–10 |
| Ventana | Anomalía en ventanas temporales procesales | 0–10 |
| Impacto | Daño mediático y político consumado | 0–10 |
| Vector | Direccionalidad adversa al objetivo (negativo) | -1–0 |

## Intensidad global

Combinación ponderada incremental de deltas por eje, con cap ±0.50 por buffer.

## Lecturas cualitativas

- &lt; 4.0: bajo
- 4.0–4.9: marginal
- 5.0–5.9: sospechas fundadas
- 6.0–6.9: alta probabilidad de lawfare
- ≥ 7.0: lawfare sistémico confirmado
