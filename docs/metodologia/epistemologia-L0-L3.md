# Epistemología L0–L3

| Capa | Descripción | Acción |
|------|-------------|--------|
| L0 | Dato bruto verificable (fecha, sentencia, URL) | Carga directa |
| L1 | Agregado con fuente | Carga peso 1.0 |
| L2 | Inferencia evidenciada | Carga peso 0.7 |
| L3 | Interpretación sin ancla | Cuarentena |
| Lx | Contradicción interna | Bloqueo hasta resolución |

Solo L0–L2 alteran los ejes. L3 aparece en informes pero no mueve la aguja.
