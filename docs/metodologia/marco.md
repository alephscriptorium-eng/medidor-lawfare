# Marco metodológico

## Hipótesis operativa

El lawfare puede detectarse mediante patrones observables en la intersección de actuaciones judiciales, cobertura mediática y calendarios políticos, sin necesidad de afirmar intencionalidad directa.

## Definición operativa

**Lawfare medido:** conjunto de indicios cuantificables (ejes) que, contextualizados con buffers históricos cribados, producen una intensidad global en escala 0–10.

## Caso foco inmutable

Cada medición se ancla a un caso concreto identificado por `caso_foco.id`. Los buffers de contexto no cambian el caso; solo normalizan la lectura.

## Falsabilidad

- Un buffer con >40% L3 no debe alterar significativamente la medición.
- Datos L0 contradictorios deben poder abrir un branch alternativo.
- La reproducción desde `estado.json` + buffers debe reproducir las mediciones de referencia del catálogo.
