Instalación

git clone {{ github_repo }}.git
cd medidor-lawfare
pip install -e ".[dev]"

Comandos CLI

medidor cribar  data/buffers/MI-BUFFER.json --caso <caso-id>
medidor commit  data/buffers/MI-BUFFER.json --caso <caso-id>
medidor catalog sync
medidor build --target all

Ciclo típico

  1. Agente externo llena buffer (llenar_buffer.md) → docs/sesiones/buffer-NN.md
  2. Operador convierte a data/buffers/MCS-N-entrada.json
  3. medidor cribar → medidor commit
  4. Si hay cuarentena L3: limpiar_cuarentena_pedir.md / limpiar_cuarentena_recibir.md
  5. medidor catalog sync + medidor build --target all

Publicar un nuevo caso

  1. Crear data/casos/{slug}/caso.json y estado.json con medición baseline
  2. medidor catalog sync
  3. medidor build --target all

Estructura public/

public/
  index.html   → índice (artefacto + centro de datos)
  prensa/      → catálogo de mediciones
  foss/        → documentación del artefacto (esta sección)
